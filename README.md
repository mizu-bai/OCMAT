# OCMAT

An Experimental Interpreter for `MATLAB` Language

一个试验性的 `MATLAB` 语言编译器（写着玩）

## Why MATLAB

写 `MATLAB` 写得挺多的，而且其语法简单，写起来应该相对容易些（心虚）

## Why OC

因为还没花精力学 `C++` （恼）

`Swift` 调 `C` 的函数不方便

## 文件结构

* `OCMAT/main.m`: 程序入口
* `OCMAT/Lexer/Lexer.*`: 词法分析类
* `OCMAT/PreProcessor/ProProcessor.*`: 预处理类
